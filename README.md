# Flexible 운영 도우미 — 익명 공개판

실제 현장 팀에서 쓰던 운영 도구의 **익명 공개판**입니다.
종이 제비뽑기 누락·재추첨으로 2~3분 걸리던 십여 명 자리배치를, 출석 확인부터 결과 공유까지
10초 이내(당시 현장 측정 기록) 흐름으로 바꿨습니다.

데모: https://flexible-seat-randomizer.vercel.app

## 원본과의 차이

이 공개판은 실제 운영에 쓰던 `index.html` 단일 파일 앱에서 **딱 두 가지만** 바꾼 것입니다.

1. 명단의 실명을 전원 가상 이름(`참여자 NN`)으로 치환
2. 페이지 설명문에 "명단은 전원 가상 이름" 표기 추가

기능·UI·코드는 현장에서 쓰던 그대로입니다. 과거 공개 배포본에서 실명 노출을 발견해
배포를 폐기했고, 이 공개판이 그 대체입니다.

> 저장소 히스토리 초반에는 React로 새로 만든 "재구축판"이 있습니다.
> 원본 소스가 보존돼 있음을 확인한 뒤, 진정성을 위해 원본 익명판으로 교체했습니다.

## 주요 기능

- **자리 배치**: 실제 사무실 배치도(TV·숫자 자리 1~13·카페팀 자리 2) 그대로의 좌석 맵
- **출석 연동**: 출석 토글에서 꺼진 인원은 배치에서 자동 제외
- **자리 잠금**: 숫자 자리를 터치하면 "사용 안 함" 처리
- **카페팀(오픈조) 자리**: 터치해서 손님 응대 인원을 지정 배치
- **결과 공유**: html2canvas 캡처로 카톡 공유용 이미지 생성
- **카페팀 출퇴근표** 탭, 다크모드, 간편 모드, 명단 수정

## 기술 노트

- 단일 `index.html` — 프레임워크·빌드 없음. 태블릿 브라우저에 바로가기로 배포해 운영
- 데이터는 `localStorage`에만 저장 (서버·DB·분석 스크립트 없음)
- 외부 의존성: Pretendard 폰트, html2canvas (CDN)

## 개인정보 원칙

- 공개판 기본 명단은 전원 가상 인물
- 입력 데이터는 사용자의 브라우저에만 저장
- 실제 구성원 이름·출결·기관 상세명은 이 저장소와 데모에 포함되지 않음
